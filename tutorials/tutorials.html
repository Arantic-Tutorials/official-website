Install VS Code

VS Code is the most widely used and versatile code editor. We will use vscode throughout this tutorial , so lets get started and install it.

To install VS code, Open this website https://code.visualstudio.com/download, After opening it follow these steps to install it

1. Select the operating system you are using right now

2. Download and install your package, After installing, You can open it! Now you will see a screen like this

3. Now, You will see a file option in the menubar, Click on that, After that you will see a New File Option, Click on it, Then, Create a new file named "index.html" and save it

4. Pro tip --> go to the extensions tab on the left of the screen and search "live server" in the search bar

this extension allow you to host your website locally and you will be able to see the live preview of your page
